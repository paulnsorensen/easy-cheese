# Foo/bar skill

Use this skill when a task specifically targets `bar` inside the `foo` scope.

## Variables

- `foo`: the parent scope.
- `bar`: the selected child value.
- `baz`: an available sibling value, only include it when comparison is useful.

## Guidance
