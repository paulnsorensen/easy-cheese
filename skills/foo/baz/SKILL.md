# Foo/baz skill

Use this skill when a task specifically targets `baz` inside the `foo` scope.

## Variables

- `foo`: the parent scope.
- `baz`: the selected child value.
- `bar`: an available sibling value, only include it when comparison is useful.

## Guidance

Summarize the relationship as `foo -> baz`, then provide the requested output.
