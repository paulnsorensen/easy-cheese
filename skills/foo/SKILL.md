# Foo skill

Use this skill when a task needs the basic easy-cheese placeholder variables.

## Variables

- `foo`: the top-level value and scope name.
- `bar`: the first child value under `foo`.
- `baz`: the second child value under `foo`.

## Guidance

When responding, keep the values simple and explicit:

1. Treat `foo` as the parent concept.
2. Treat `bar` and `baz` as sibling child concepts.
3. Prefer plain text output unless the user asks for another format.
